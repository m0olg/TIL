### .self
# 01 `.self`란?
→ 그 타입 또는 자기 자신을 그대로 가리키는 것으로 `ForEach`나 `List`에서 데이터 배열의 각 요소를 고유 식별자(ID)로 지정할 때 사용 됨

# 02 메타타입의 값을 얻어내기 위한 `.self`


# 03 `ForEach`에서 `\.self`를 사용하는 이유
SwiftUI에서 `ForEach`는 `Array` 등 컬렉션 자료형을 순환하며 `View`를 생성해주는 역할을 함 \
기존의 `for-in` 반복문과 비슷해 보이지만, SwiftUI의 `forEach`는 단순 반복이 아니라 **각 요소를 화면으로 변환**하는 구조라는 점에서 차이가 있음 \
또한 `ForEach`는 각 데이터를 구분하기 위해 반드시 `id`가 필요하며 이 값은 View를 효율적으로 업데이트 하는데 사용 됨

### 이유
본론으로 들어와서 왜 쓰는지 이유를 알아보자면

SwiftUI가 **각 요소를 구분하기 위해 ID가 필요하기 때문**임 \
`\.self`의 의미가 각 요소 자체를 ID로 쓰겠다는 말이기 때문에 만약 ID가 없으면 스유가 혼란이 옴 (이게 누가 누군지)

정리하면 \
**`ForEach`에서 `\.self`는 값 자체를 ID로 써서 각 요소를 구분하려고 쓰는 것**임

# 04 쓰는 조건
- 값이 중복되지 않을때
- 값 자체가 고유할 때

위 같은 상황일 때 쓸 수 있다 ‼️
