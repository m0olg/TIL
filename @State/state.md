# 01 `@State`란?
→ 화면이 바뀌게 만드는 "값 저장 버튼" 이라고 할 수 있음 (이 프로퍼티가 하고 싶은 행동을 정의하는 타입) \
= 값이 바뀌면 화면도 다시 그림

# 02 코드로 보면
```swift
@State private var count = 0
```
이게 뭔 의미냐면
- `count`라는 숫자를 저장하고
- 이 값은 **특별한 상태 값**
#### 그리고 이제 버튼을 누르면 ⬇️
```swift
Botton("++") {
    count += 1
}
```
- 버튼을 클릭하면
- `count` = 1이 됨
- 그리고 SwiftUI가 자동으로 화면을 다시 그림

### ⭐️⭐️
`@State`는 일반변수와는 다름 (상태변수라고 생각하면 될 듯 하다)

**일반 변수**는 값이 바뀌어도 화면은 변하지 않지만 \
**상태 변수**(`@State`)는 값이 바뀌면 화면도 자동 업데이트 됨

---
### 한 줄로 요약하면
상태(`@State`) → 값이 바뀌면 → 자동으로 화면 리렌더링

`@State` = 상태를 지정하는 상태 변수
