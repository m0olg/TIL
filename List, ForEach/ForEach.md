### ForEach

# 01 `ForEach`이란?
→ 배열 같은 데이터 묶음을 하나씩 꺼내서 각 데이터마다 View를 만들어주는 역할을 하며 동작 List처럼 값을 넣어줘야 함 (데이터 개수만큼 화면 요소를 자동 생성해준다고 생각하면 쉬움)

# 02 기본 문법
ForEach(데이터, id: \.고유값) { item in
    View 코드
}

# 03 `for - in`과 `forEach`의 차이점

- 위 둘 다 반복을 돌지만 역할과 사용 방식이 조금 다름

|        | 형태 | 사용 방식 | break/continue | return |
|--------|------|----------|:----------------:|--------|
| for-in | 문법 (언어 기능) | 반복문 | ⭕️ | 바깥 함수에 영향 |
| forEach | 메서드 (함수) | 컬렉션 함수 | ❌ | 클로저만 종료 |

# 04 특징
- `ForEach` = 데이터를 View로 변환
- SwiftUI에서 반복 UI를 만들 때 씀
- 각 요소를 구분하기 위해 id 필요
- `for - in`문과 역할이 다름
