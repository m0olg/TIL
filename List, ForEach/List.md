### List로 정적 컨텐츠 표현하기
```swift
List {
    Text("hello")
    Text("world")
}
```
뷰 빌더의 제약 때문에 정적 컨텐츠 표현시 중괄호 안에 10개 넘는 뷰를 집어넣으면 에러 뜸

### List로 통적 컨텐츠 표현하기
(동적 컨텐츠란? \
실행 중에 '데이터에 따라 바뀌는 UI' = 개수나 내용이 정해져 있지 않은 뷰)
```swift
List(0..<17) { index in
	Text("\(index)")
}
```
- RandomAccessCollention 프로토콜을 준수하는 데이터를 집어넣을 수도 있는데 이 경우 각 요소들을 식별할수 있도록 id 값을 제공해야함
